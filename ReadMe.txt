# Projet PHP - Application de gestion d'une équipe de football

## URL du site
http://oumerretane-corbille-football.infinityfreeapp.com/

## Description
Ce projet est une application web complète permettant aux entraîneurs de gérer leur équipe sportive, les matchs, les compositions d'équipes et les statistiques des joueurs. Le site a été développé en PHP avec une base de données MySQL.

## Informations d'authentification
Pour accéder à l'application, utilisez les identifiants suivants :
- Nom d'utilisateur : monsieur.entraineur@club.com
- Mot de passe : entraineur123

## Fonctionnalités
- Gestion des joueurs (ajout/modification/suppression/accès aux informations)
- Gestion des matchs (création/modification/suppression/accès aux informations)
- Configuration des feuilles de match
- Évaluation après le match
- Statistiques (par matchs ou par joueurs)

## Utilisation
Ce site est hébergé par InfinityFree.  
Pour y accéder, il suffit de cliquer sur l'URL du site et d'entrer les informations d'authentification.

## Base de données
Base MySQL contenant les tables : joueurs, matchs, commentaire, entraineur, participer

## Installation
Aucune installation nécessaire, l'application est accessible en ligne via l'URL.

## Technologies
- PHP
- MySQL (à travers PDO et PHPMyAdmin)
- HTML/CSS
